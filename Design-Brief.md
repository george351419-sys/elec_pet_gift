# Design Brief：魔法宠物（Magic Pet）

**状态**：已定稿（作为最高视觉权威，无独立设计稿）
**最后更新**：2026-06-23

---

## 1. 产品形态

**移动优先 Web 应用**，单页，两个 Tab 切换。

- 主体宽度：max-width 480px，水平居中
- 背景：全屏渐变色，内容区卡片式浮起
- 不需要响应式多列布局——单栏到底，PC 上看像手机 App

**两个使用者，两种状态，同一个 URL：**
- Tab 1（家长设置）：信息密度高，需要填写和确认，偏"工具感"
- Tab 2（宠物陪伴）：信息密度极低，大图、大字、大按钮，偏"沉浸感"

---

## 2. 体验原则

1. **孩子进入 Tab 2 之后，不应该看到任何让他困惑的东西。** 只有宠物、字幕、一个大按钮。
2. **家长的设置流程不能让人觉得在"填表"。** 访谈是主入口，表格是辅助确认。
3. **宠物是有生命的，不是播放器。** idle 时轻微呼吸，说话时嘴动，不要静止图片。
4. **反馈要即时可见。** 说话时字幕滚动，音量有可视指示，连接状态文字说清楚。
5. **颜色和圆角传递安全感。** 没有尖角，没有高饱和，不像医疗也不像游戏广告。

---

## 3. 信息架构

```
App Shell
├── Header（Logo + Tab 切换）
├── Tab 1：家长设置
│   ├── 状态摘要行（当前宠物 + 孩子昵称 + 档案完整度）
│   ├── 访谈区块（FullDuplexVoice 组件，可折叠）
│   ├── 孩子档案区块（结构化字段，可编辑）
│   └── 宠物选择区块（5 张卡片，横向滚动）
└── Tab 2：宠物陪伴
    ├── 宠物昵称（居中，顶部）
    ├── 宠物形象（居中，占视窗 50%）
    ├── 宠物字幕（大字，宠物说的话）
    ├── 孩子字幕（小字，孩子说的话）
    └── 操作按钮区（开始 / 静音 / 打断 / 结束）
```

## 4. 视觉方向

### 4.1 调色板

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景渐变起点 | `#FFF8F0`（米白暖） | 全屏背景顶部 |
| 背景渐变终点 | `#EDE7FF`（淡紫） | 全屏背景底部 |
| 主色 | `#7C5CBF`（温暖紫） | Tab 激活态、主按钮、accent |
| 主色浅版 | `#E8DFFF` | 选中卡片背景、hover 态 |
| 辅色 | `#FF8C69`（珊瑚橙） | 危险按钮（结束）、警示 |
| 字幕-宠物 | `#3D2B6B`（深紫） | 宠物说话字幕 |
| 字幕-孩子 | `#888888`（中灰） | 孩子说话字幕 |
| 卡片背景 | `#FFFFFF` with `box-shadow: 0 4px 20px rgba(124,92,191,0.12)` | 所有白色浮起卡片 |
| 禁用态 | `#CCCCCC` | 未配置时的按钮 |

### 4.2 字体

- **字体族**：优先使用系统圆体——`'PingFang SC', 'HarmonyOS Sans', 'Noto Sans SC', sans-serif`
- **行高**：1.6（内容），1.2（按钮/标签）
- **字号体系**：

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 宠物字幕 | 22px | 600 | Tab 2 宠物说的话 |
| 正文 | 15px | 400 | 档案字段内容 |
| 标签/按钮 | 14px | 500 | 按钮文字、字段标签 |
| 辅助说明 | 12px | 400 | 状态提示、孩子字幕 |

### 4.3 圆角与间距

- 卡片圆角：`border-radius: 20px`
- 按钮圆角：`border-radius: 999px`（胶囊形）
- 输入框圆角：`border-radius: 12px`
- 间距基础单位：8px（8/16/24/32px 四档）
- 页面内边距：16px（左右）

### 4.4 动效

- Tab 切换：`opacity` + `transform: translateY(8px)` 淡入，`150ms ease-out`
- 宠物 idle 动画：`@keyframes breathe { 0%,100% { transform: scale(1) } 50% { transform: scale(1.03) } }` 3s 循环
- 宠物说话动画：根据 `remoteLevel` 数值动态设置 `--mouth-open: 0~1`，驱动嘴部 PNG 帧切换（0=闭嘴，1=大开）
- 按钮 hover：`transform: scale(1.03)` + `box-shadow` 加深，`100ms`
- 字幕出现：新一轮字幕从下方 `translateY(6px)` 淡入

## 5. 组件与状态

### 5.1 Tab 切换器

```
[家长设置]  [宠物陪伴]
```
- 两个 Tab，等宽，居中
- 激活态：下划线 + 主色文字，`font-weight: 600`
- 未激活：灰色，`font-weight: 400`
- 切换动画：内容区淡入淡出

### 5.2 语音访谈组件（Tab 1 内嵌 FullDuplexVoice）

- idle 态：显示"开始语音访谈"按钮（主色胶囊按钮，全宽）
- 连接中：spinner + 状态文字
- 已连接：
  - 顶部：AI 头像 + 状态文字 + 计时器
  - 中部：音量波形（简单 bar）
  - 底部：[静音] [打断] [结束访谈] 三个按钮
  - 字幕列表：最近 4 轮，role 区分颜色
- 结束态：折叠为"继续补充"链接文字按钮

### 5.3 档案字段组件

- 每个字段：上方 `label`，下方 `input/textarea`
- `input`：圆角 12px，border `1px solid #E0D8F5`，focus 时 border 变主色
- `textarea`（亲子回忆）：最小高度 80px，自动撑高
- 性格标签：多选 chip，选中态背景 `#E8DFFF` + 主色边框
- 整体用卡片包裹，内边距 16px

### 5.4 宠物选择卡片

- 横向滚动列表（`overflow-x: auto`，隐藏滚动条）
- 每张卡片：宽 140px，圆角 16px，白色背景
  - 上：宠物 PNG 图（正方形 100px）
  - 中：宠物类型名（14px）
  - 下：一句性格描述（12px 灰色）
- 选中态：`border: 2px solid #7C5CBF` + `background: #EDE7FF`
- 底部"确认选择"按钮：主色胶囊，全宽，未选时禁用灰

### 5.5 宠物形象区（Tab 2 核心）

- 容器：圆形背景光晕 `radial-gradient`（柔和白→透明），直径 240px
- 宠物 PNG：200px × 200px，`object-fit: contain`
- 说话态：JS 根据 `remoteLevel` 每 100ms 切换 PNG 帧（idle_0.png / talking_1.png / talking_2.png）
- idle 态：`animation: breathe 3s ease-in-out infinite`
- 宠物昵称：宠物图上方 16px，`font-size: 18px font-weight: 600`，主色

### 5.6 字幕区

- 宠物字幕：`font-size: 22px`，颜色 `#3D2B6B`，居中，最多 2 行，超出省略
- 孩子字幕：`font-size: 13px`，颜色 `#888`，居中，前缀"你说："
- 两者之间间距 12px
- 新字幕淡入，旧字幕不消失（保留上一条，透明度降至 0.4）

### 5.7 操作按钮区

**idle / ready 态：**
```
[  开始对话（全宽主色胶囊）  ]
```

**connected 态：**
```
[静音]  [打断]  [结束对话（辅色）]
```
- 静音/打断：次级按钮（白色背景 + 主色边框）
- 结束：危险按钮（珊瑚橙背景，白色文字）
- 三按钮等宽排一行，间距 8px

**未配置 / 错误态：**
```
[  请先完成家长设置（禁用，灰色）  ]
```
下方 12px 小字说明原因

## 6. 空/加载/错误状态

| 状态 | 展示方式 |
|------|----------|
| Tab 2 无档案 | 宠物区显示虚线圆形占位 + "等家长来设置～" 文案，按钮禁用 |
| 宠物图加载中 | 主色 spinner 在 200px 圆区域居中 |
| 连接中 | 状态文字动态显示"正在唤醒 {宠物名}…" |
| 麦克风权限拒绝 | 卡片内显示图标 + 说明文字 + "去设置" 链接按钮 |
| RTC 断线 | Toast 提示（顶部弹出，3s 自动消失）+ 重连按钮 |

## 7. 宠物 PNG 资产规格

每种宠物需要以下帧（共 3 帧 × 5 种 = 15 张图）：

| 文件名规则 | 说明 |
|-----------|------|
| `{type}_idle.png` | 静止态（嘴闭，表情自然） |
| `{type}_talking_1.png` | 说话态轻（嘴微开） |
| `{type}_talking_2.png` | 说话态重（嘴大开） |

- 尺寸：400×400px（2x 显示，实际渲染 200px）
- 背景：透明 PNG
- 风格：3D 立体感，圆润，柔和阴影，颜色饱和度中等偏低（不刺眼）
- 来源：本期 demo 使用 AI 生成或外购免版权素材

---

## 8. 设计决策 vs Spec 差异

本次 Design Brief 未改变 Spec 中的任何功能决策，以下 UI 细节为新增明确：

- 宠物说话动画采用 3 帧 PNG 切换，而非 CSS 变形
- 字幕保留上一条（透明度 0.4），不完全消失，帮助孩子理解对话上下文
- 宠物选择卡片横向滚动，不分行排列（移动端更自然）
- 操作按钮区 idle/connected/disabled 三态完整定义
